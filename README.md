# cctv-jiuyou-archive

本项目用于归档和发布多个独立的 HTML 页面。这些页面可能来源于不同的项目、工具或演示用途，彼此之间没有强关联。仓库的主要目的是提供一个结构化的存档空间，方便访问、引用或历史查阅。

## 项目简介

本仓库不针对任何特定网站或服务，仅作为 HTML 静态页面的集合。所有页面均为独立文件，可直接通过浏览器打开或部署到静态托管服务。内容涉及多种类型，包括但不限于信息展示、工具界面、实验性页面等。

## 目录说明

```
/
├── index.html          # 仓库首页（可选）
├── page1/              # 示例子目录
│   └── index.html
├── page2/              # 示例子目录
│   └── index.html
└── README.md           # 本文件
```

每个子目录通常包含一个或多个 HTML 页面及其相关资源（如 CSS、JavaScript、图片等）。目录名称一般反映页面功能或主题，但并非严格分类。

## 页面归档说明

- 所有页面均为静态 HTML，无后端依赖。
- 部分页面可能包含外部资源（如 CDN 库），但尽量保证离线可查看。
- 页面内容可能随时间更新或调整，不保证永久不变。
- 归档目的为保留原始形态，通常不对内容做额外修饰或优化。

## 使用方式

你可以直接通过 GitHub Pages 或其他静态托管服务部署本仓库。例如：

1. 克隆或下载本仓库。
2. 将整个目录上传到任意静态服务器。
3. 访问对应的 HTML 文件路径即可。

如需本地预览，直接在浏览器中打开 HTML 文件即可（部分页面可能需要本地服务器支持跨域资源）。
